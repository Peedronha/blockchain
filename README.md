# Simple Java Blockchain ⛓️

A minimalist and conceptual implementation of a Blockchain developed in Java. This project demonstrates the core fundamentals of how blocks are securely chained together and how the **Proof of Work (PoW)** consensus mechanism operates using the SHA-256 cryptographic algorithm.

---

## 🚀 Features

* **Cryptographic Hash Generation:** Uses SHA-256 to ensure data integrity for every single block.
* **Secure Chaining:** Each block contains the hash of the previous block, rendering the chain immutable.
* **Proof of Work (Mining):** A simulated mining mechanism that requires computational power to find a valid hash based on a dynamic difficulty target.

---

## 🧠 How It Works

### Block Structure
Every block in the network stores the following properties:
1.  **Data (`data`):** The actual information or transaction records stored inside the block.
2.  **Previous Hash (`previousHash`):** The cryptographic link that connects the current block to its predecessor.
3.  **Timestamp (`timeStamp`):** The exact time (in milliseconds) when the block was created.
4.  **Nonce:** A counter variable used exclusively for the mining process.
5.  **Current Hash (`hash`):** The unique digital signature of the block, generated by combining all the fields listed above.
